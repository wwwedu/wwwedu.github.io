What This Notebook Covers

This notebook accompanies Time Series Part 2, covering the tools needed to handle the most common pathologies in economic and financial time series. The central problem is non-stationarity: when a series has no stable mean or variance, standard OLS inference collapses entirely, and applying OLS to trending series produces spurious regressions with high \(R^2\) and significant \(t\)-statistics that are economically meaningless.

We’ll cover:

  • Three types of non-stationarity and why the correct remedy differs for each
  • Unit root testing: Dickey-Fuller, ADF, Phillips-Perron, KPSS
  • Spurious regression: what it looks like and why it happens
  • Cointegration: when two I(1) series share a genuine long-run relationship
  • ARIMA modelling from scratch: identification, estimation, diagnostics, forecasting
  • Seasonal ARIMA (SARIMA) using the classic AirPassengers dataset
  • Bonus: Arellano-Bond dynamic panel GMM

2 Non-Stationarity and Unit Roots

2.1 Motivating Example: GDP and Inflation

Before examining interest rates, consider two core macroeconomic series from the usa dataset: real GDP and inflation (change in CPI). Both are textbook examples of I(1) processes — they wander without a stable mean, but their first differences are stationary.

data("usa")
usa.ts <- ts(usa, start = c(1984, 1), end = c(2009, 4), frequency = 4)

# Build first differences
Dgdp <- diff(usa.ts[, "gdp"])
Dinf <- diff(usa.ts[, "inf"])
Df   <- diff(usa.ts[, "f"])
Db   <- diff(usa.ts[, "b"])

usa.ts.df <- ts.union(
  gdp = usa.ts[, "gdp"],
  inf = usa.ts[, "inf"],
  f   = usa.ts[, "f"],
  b   = usa.ts[, "b"],
  Dgdp, Dinf, Df, Db,
  dframe = TRUE
)

options(repr.plot.width = 10, repr.plot.height = 8, repr.plot.res = 150)
par(mfrow = c(2, 2))
plot(usa.ts.df$gdp,  main = "GDP")
plot(usa.ts.df$Dgdp, main = "GDP: First Difference")
plot(usa.ts.df$inf,  main = "Inflation (Change in CPI)")
plot(usa.ts.df$Dinf, main = "Inflation: First Difference")

What to notice: Both GDP and inflation drift persistently over time with no tendency to revert to a fixed mean — the hallmark of a non-stationary, I(1) process. Once we take first differences, however, both series fluctuate around a stable mean: \(\Delta\text{GDP}_t\) and \(\Delta\Delta\text{CPI}_t\) are visually consistent with stationarity. This is the core motivation for unit root testing and the Box-Jenkins approach to ARIMA modelling — work in differences, not levels.

2.3 Three Types of Non-Stationarity

The lecture distinguishes three data-generating processes that all have a unit root but require different fixes:

Type DGP Fix
Random walk \(y_t = y_{t-1} + \varepsilon_t\) First difference
Random walk with drift \(y_t = \mu + y_{t-1} + \varepsilon_t\) First difference
Deterministic trend \(y_t = \mu + bt + \varepsilon_t\) Detrend

Key asymmetry: Applying first-differencing to a trend-stationary series introduces a spurious MA(1) component (over-differencing). Detrending a random walk leaves the stochastic trend intact — the residuals are still non-stationary.

N   <- 500
rho <- 0.7
a   <- 1
l   <- 0.01

set.seed(246810)
v <- ts(rnorm(N, 0, 1))

par(mfrow = c(2,3))

# 1. Stationary about zero
y <- ts(rep(0, N))
for (t in 2:N) y[t] <- rho * y[t - 1] + v[t]
plot(y, type = "l", main = c("Stationary about zero:", "y[t] = 0.7*y[t-1] + v[t]"))
abline(h = 0)

# 2. Stationary about non-zero mean
y <- ts(rep(0, N))
for (t in 2:N) y[t] <- a + rho * y[t - 1] + v[t]
plot(y, type = "l", main = c("Stationary about non-zero:", "y[t] = 1 + 0.7*y[t-1] + v[t]"))
abline(h = 0)

# 3. Stationary about deterministic trend
y <- ts(rep(0, N))
for (t in 2:N) y[t] <- a + l * time(y)[t] + rho * y[t - 1] + v[t]
plot(y, type = "l", main = c("Stationary about deterministic trend:", "y[t] = 1 + t + 0.7*y[t-1] + v[t]"))
abline(h = 0)

# 4. Random walk
y <- ts(rep(0, N))
for (t in 2:N) y[t] <- y[t - 1] + v[t]
plot(y, type = "l", main = c("Random walk:", "y[t] = y[t-1] + v[t]"))
abline(h = 0)

# 5. Random walk with drift
y <- ts(rep(0, N))
for (t in 2:N) y[t] <- 0.1 + y[t - 1] + v[t]
plot(y, type = "l", main = c("Random walk with drift:", "y[t] = 0.1 + y[t-1] + v[t]"))
abline(h = 0)

# 6. Random walk about deterministic trend
y <- ts(rep(0, N))
for (t in 2:N) y[t] <- 0.1 + l * time(y)[t] + y[t - 1] + v[t]
plot(y, type = "l", main = c("Random walk about deterministic trend:", "y[t] = 0.1 + t + y[t-1] + v[t]"))
abline(h = 0)
par(mfrow = c(1, 1))

Note the similarity between the deterministic trend stationary process (top-right) and segments of the random walk (middle-left). Because shocks accumulate without bound in the random walk, the series exhibits long pseudo-trends — this is why we call them processes with a “stochastic trend.”

2.4 The Augmented Dickey-Fuller Test

2.4.1 Setup

The DF test reparameterises the AR(1) as:

\[\Delta Y_t = \beta_0 + \delta Y_{t-1} + u_t, \qquad \delta \equiv \phi - 1\]

Test: \(H_0: \delta = 0\) (unit root) vs. \(H_1: \delta < 0\) (stationary). The ADF augments with lagged differences to absorb any higher-order serial correlation:

\[\Delta Y_t = \beta_0 + \delta Y_{t-1} + \gamma_1 \Delta Y_{t-1} + \cdots + \gamma_p \Delta Y_{t-p} + u_t\]

Critical values are non-standard (biased left relative to \(t\) distribution) and must be obtained by simulation.

We work with the 3-month T-bill rate (tbill) as our reference series — the same series used in the lecture.

test.ts <- tbill

options(repr.plot.width = 6, repr.plot.height = 6, repr.plot.res = 150)
par(mfrow = c(1, 1))
plot(test.ts, col = "black", lwd = 2,
     ylab = "%",
     xlab = "Date",
     main = "TBill Rate")
options(repr.plot.width = 6, repr.plot.height = 6, repr.plot.res = 150)
par(mfrow = c(1, 1))
acf(test.ts, main = "Sample Autocorrelation for Tbill Data")
par(mfrow = c(1, 1))
pacf(test.ts, main = "Sample Partial Autocorrelation for Tbill Data")

The ACF decays very slowly and barely reaches zero — a textbook signature of a near-unit-root process. The PACF has a large spike at lag 1 with rapid decay, consistent with an AR(1) or unit root.

Note on KPSS: Unlike ADF/PP, the null hypothesis of KPSS is stationarity. A test statistic that exceeds the critical value leads us to reject stationarity. Use it as a complement to ADF rather than a replacement.

3.2 Spurious Regression Simulation

4 Cointegration

4.1 What Cointegration Means

Two I(1) series \(y_t\) and \(x_t\) are cointegrated if there exists \(\beta\) such that:

\[y_t - \beta x_t \sim I(0)\]

The linear combination is stationary even though both series individually wander. Economically, this means the two series share a common stochastic trend — they can deviate in the short run but cannot drift apart permanently.

Engle-Granger two-step procedure:

  1. Regress \(y_t\) on \(x_t\) (the cointegrating regression)
  2. Test the residuals \(\hat{u}_t\) for a unit root using ADF or PP

If residuals are stationary → cointegration. Critical values are more negative than standard ADF critical values (non-standard, depends on number of variables).

5 Univariate Time Series: ARIMA

5.1 ACF and PACF: Reading the Correlogram

Before fitting an ARIMA model, we use the autocorrelation function (ACF) and partial autocorrelation function (PACF) to identify plausible model orders. The identification table from the lecture:

Model ACF PACF
AR(\(p\)) Decays geometrically Cuts off sharply at lag \(p\)
MA(\(q\)) Cuts off sharply at lag \(q\) Decays geometrically
ARMA(\(p,q\)) Decays after lag \(q\) Decays after lag \(p\)
I(1) (unit root) Barely decays, near 1 Large spike at lag 1 only
White noise All near zero All near zero

In practice, ACF/PACF patterns are rarely textbook-clean. Use them as a starting point for candidate models, then select using AIC/BIC.

6 Dynamic Panels: Arellano-Bond (Bonus)

When the panel model itself includes a lagged dependent variable:

\[y_{it} = \rho y_{i,t-1} + \mathbf{x}_{it}'\boldsymbol{\beta} + \mu_i + \varepsilon_{it}\]

the within (FE) transformation does not solve endogeneity because \(y_{i,t-1} - \bar{y}_i\) remains correlated with the demeaned error. Arellano-Bond (1991) addresses this with first-differencing to eliminate \(\mu_i\), then using deeper lags of \(y_{it}\) as GMM instruments for the differenced lagged variable.

We illustrate using the EmplUK dataset (Arellano-Bond’s original application): employment dynamics for 140 UK firms, 1976–1984.

6.0.1 Check for serial correlation in FE residuals

Key diagnostics:

  • Sargan-Hansen test: tests overidentifying restrictions. Failure to reject → instruments appear valid.
  • AR(1) test: first-order autocorrelation in first-differenced residuals is expected by construction — not a problem.
  • AR(2) test: second-order autocorrelation would invalidate the lag-2 instruments. We want to fail to reject here.

Important caveat: Arellano-Bond is sensitive to the choice of instrument lag range. Proliferating instruments (using too many lags) weakens the Sargan test and can bias GMM estimates. Always run robustness checks with different lag depths.

7 Summary and Key Takeaways

7.1 The Non-Stationarity Checklist

  1. Plot the series — does it trend? Does it mean-revert?
  2. Test for structural breaks (Quandt-Andrews/Bai-Perron) if suspected
  3. Run ADF (with trend if the series trends)
    • Reject \(H_0\) → stationary, work in levels
    • Fail to reject → proceed
  4. Determine type: stochastic vs deterministic trend
  5. Apply correct fix: difference I(1) series; detrend trend-stationary series
  6. If multiple I(1) series: test cointegration before differencing — you may throw away a real long-run relationship

7.2 ARIMA Box-Jenkins Workflow

Stage What to do
Identification Plot, unit root test, determine \(d\); ACF/PACF for \(p\), \(q\)
Estimation Estimate candidates via ML; compare AIC/BIC
Diagnostics Ljung-Box on residuals; ACF/PACF of residuals
Forecasting Generate forecasts + intervals; evaluate out-of-sample

ARIMA selection is iterative, not linear: cycle back through identification and estimation if diagnostics fail.
