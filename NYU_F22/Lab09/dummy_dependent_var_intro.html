Regression with Binary Dependent Variables

Instead of using a continuous regressor $X$, we might be interested in running the regression $$ Y_{i}=\beta_{0}+\beta_{1} D_{i}+u_{i} $$ where $D_{i}$ is a binary variable, a so-called dummy variable.

We may, for example, be interested in whether student performance is impacted by class size. The California Test Score dataset (available via the AER package) includes average test scores and average students-teacher-ratio (STR) on 420 California school districts for the 1998-1999 school year. Using a dummy variable we may split the 420 districts into large vs. small class size districts $$ D_{i}= \begin{cases}1 & \text { if } S T R \text { in } i^{t h} \text { school district }<20 \\ 0 & \text { if } S T R \text { in } i^{t h} \text { school district } \geq 20\end{cases} $$

and estimate the model $$ \text { TestScore }_{i}=\beta_{0}+\beta_{1} D_{i}+u_{i} $$

Let's begin by loading and setting up our data:

With $D$ as the regressor, it is not useful to think of $\beta_{1}$ as a slope parameter since $D_{i} \in\{0,1\}$, i.e., we only observe two discrete values instead of a continuum of regressor values. There is no continuous line depicting the conditional expectation function $E\left(\right.$ TestScore $\left._{i} \mid D_{i}\right)$ since this function is solely defined for $x$-positions 0 and 1 . Therefore, the interpretation of the coefficients in this regression model is as follows:

  • $E\left(Y_{i} \mid D_{i}=0\right)=\beta_{0}$, so $\beta_{0}$ is the expected test score in districts where $D_{i}=0$ where $S T R$ is above 20
  • $E\left(Y_{i} \mid D_{i}=1\right)=\beta_{0}+\beta_{1}$ or, using the result above, $\beta_{1}=E\left(Y_{i} \mid D_{i}=1\right)-E\left(Y_{i} \mid D_{i}=0\right)$. Thus, $\beta_{1}$ is the difference in group-specific expectations, i.e., the difference in expected test score between districts with $S T R<20$ and those with $S T R \geq 20$

We will now use R to estimate the dummy regression model:

The vector D has the type logical and is shown in the regression summary: the label DTRUE states that all entries TRUE are coded as 1 and all entries FALSE are coded as 0. Thus, the interpretation of the coefficient DTRUE is as stated above for $\beta_{1}$.

We can see that the expected test score in districts with $S T R<20\left(D_{i}=1\right)$ is predicted to be $650.1+7.17=657.27$ while districts with $S T R \geq 20\left(D_{i}=0\right)$ are expected to have an average test score of only $650.1$.

We can use the function predict( ) to obtain estimates of the group specific means. The red dots represent these sample group averages. Accordingly, $\hat{\beta}_{1}=7.17$ can be seen as the difference in group averages.

The model summary also answers the question whether there is a statistically significant difference in group means. This in turn would support the hypothesis that students perform differently when they are taught in small classes. We can assess this by a two-tailed test of the hypothesis $H_{0}: \beta_{1}=0$. Conveniently, the $t$ statistic and the corresponding $p$-value

Since $t$ value $=3.88>1.96$ we reject the null hypothesis at the $5 \%$ level of significance. The same conclusion results when using the $p$-value, which reports significance up to the $0.00012 \%$ level. As done with linear_model, we may alternatively use the function confint () to compute a $95 \%$ confidence interval for the true difference in means and see if the hypothesized value is an element of this confidence set.
